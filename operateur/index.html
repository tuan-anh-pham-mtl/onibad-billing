    Guide de l'opérateur¶

    Ce guide est votre référence principale pour gérer les inscriptions et la facturation. Gardez-le à portée de main. Chaque section est une tâche complète.

    Rappel

    Ce système est utilisé environ 2 fois par an (automne et hiver). Il est normal de devoir relire le guide à chaque saison.

    Résumé des actions courantes¶

    Situation Menu à utiliser Onglet actif
    Nouvelle inscription à réviser 📋 Inscriptions → Réviser Inscriptions
    Accepter un joueur 📋 Inscriptions → ✅ Accepter Inscriptions
    Mettre en liste d'attente 📋 Inscriptions → ⏳ Liste d'attente Inscriptions
    Refuser un joueur 📋 Inscriptions → ❌ Refuser Inscriptions
    Envoyer la notification 📋 Inscriptions → 📨 Finaliser Inscriptions
    Créer la facture 💰 Facturation → 📝 Générer Inscriptions
    Facture pour une famille 💰 Facturation → 👪 Familiale Inscriptions
    Joueur a payé Modifier directement la cellule Suivi facturation
    Joueur annule 💰 Facturation → 🚫 Abandonné Suivi facturation
    Remboursement 💰 Facturation → 💸 Note de crédit Suivi facturation
    Inscription par téléphone 🔧 Outils → ➕ Manuelle N'importe quel onglet
    Corriger un nom/email Modifier dans Répertoire, puis 🔧 → 🔄 Propager Répertoire joueurs

    Sections du guide¶

    • Démarrage rapide — le flux normal de A à Z
    • Réviser les inscriptions — accepter, refuser, mettre en attente
    • Facturation — générer les factures, facturation familiale, paiements
    • Liste d'attente — promouvoir, fermer
    • Annulations et remboursements — abandon, notes de crédit, régénération
    • Outils et corrections — inscription manuelle, correction de données, organisation Drive
    • Nouvelle session — checklist de début de session
    • Référence — statuts, onglets, courriels automatiques
    • Dépannage — problèmes courants

    Ouvrir le système¶

    1. Ouvrez le classeur Google Sheets de facturation
    2. Le menu 🏸 Accès Badminton apparaît dans la barre de menus (en haut)
    3. Si le menu n'apparaît pas, rechargez la page (F5)
