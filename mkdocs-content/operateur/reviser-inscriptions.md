# Réviser les inscriptions

## Quand ?

Après qu'un joueur a soumis le formulaire d'inscription.

## Comment ça fonctionne

Le système crée automatiquement des lignes dans l'onglet **Inscriptions** (une ligne par programme choisi). Le statut initial est **En révision**.

---

## Étapes

1. Allez à l'onglet **Inscriptions**
2. Menu → **📋 Inscriptions → 📋 Réviser les inscriptions**
    - Un résumé s'affiche avec tous les joueurs en attente de révision
3. **Cliquez sur la ligne** de l'inscription à traiter
4. Choisissez l'action :
    - **✅ Accepter** : le joueur est accepté pour ce programme
    - **⏳ Liste d'attente** : le programme est complet, le joueur attend
    - **❌ Refuser** : le joueur ne peut pas participer (vous devrez donner un motif)

---

## Si un joueur a choisi plusieurs programmes

Chaque programme est une ligne séparée. Vous pouvez prendre des décisions différentes par programme :

| Programme | Décision |
|-----------|----------|
| Entraînement Adulte Jeudi | ✅ Accepter |
| Entraînement Junior Dimanche | ⏳ Liste d'attente (complet) |

---

## Accepter tout le monde d'un coup

Si vous souhaitez accepter toutes les inscriptions en attente :

Menu → **📋 Inscriptions → ✅ Accepter TOUTES**

---

## Doublons

!!! warning "Doublons détectés"
    Si le système détecte qu'un joueur a soumis le formulaire deux fois pour les mêmes programmes, les lignes seront marquées **⚠️ Soumission en double**. Supprimez les doublons manuellement.

---

## Finaliser et notifier le joueur

**Quand** : Après avoir pris une décision sur **tous** les programmes d'une soumission.

Cette action envoie **un seul courriel** au joueur avec le résumé de toutes les décisions.

### Étapes

1. **Toutes les inscriptions** de la même soumission doivent avoir un statut (aucune en « En révision »)
2. Cliquez sur **n'importe quelle ligne** de la soumission
3. Menu → **📋 Inscriptions → 📨 Finaliser la révision**
4. Un résumé s'affiche. Confirmez.
5. Le système crée une ligne dans l'onglet **Notifications**
6. **Autocrat** envoie le courriel dans l'heure

### Que reçoit le joueur ?

Un seul courriel listant chaque programme et la décision :

- « Entraînement Adulte Jeudi → **Accepté** (facture à suivre) »
- « Entraînement Junior Dimanche → **En attente** (nous vous contacterons si une place se libère) »
