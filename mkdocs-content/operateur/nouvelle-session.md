# Début de nouvelle session

À faire **2 fois par an** (automne et hiver).

---

## Checklist

### 1. Mettre à jour la session courante

- Onglet **Configuration** → cellule `session_courante` → changez à la nouvelle session (ex : `A2026`)

### 2. Mettre à jour les tarifs

- Onglet **Tarifs** → mettez les anciens tarifs à `Non` dans la colonne **Actif**
- Ajoutez les nouveaux tarifs avec les bons prix, codes articles et session

### 3. Réinitialiser le compteur de factures (optionnel)

- Onglet **Configuration** → `prochain_no_facture` → remettez à `1` si vous le souhaitez

### 4. Mettre à jour le formulaire Google

- Modifiez les choix d'activité dans le formulaire pour refléter la nouvelle session

### 5. Vérifier la configuration

- Menu → **🔧 Outils → ⚙️ Vérifier la configuration**
- Corrigez tout élément marqué ❌

### 6. Organiser les PDF de l'ancienne session

- Menu → **🔧 Outils → 📁 Organiser les factures (Drive)**

!!! tip "Conseil"
    Faites un export ou une copie de sauvegarde du classeur avant de démarrer une nouvelle session. Cela simplifie les audits futurs.
